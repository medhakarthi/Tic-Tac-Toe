# Tic Tac Toe (Pygame)

A simple **beach-themed Tic Tac Toe** game built with **Python + Pygame**.  
Play as a **Starfish** vs **Seashell** on a custom tic-tac-toe board background. The game automatically detects **wins** and **draws**, displays a message, then **resets** for a new round.

---

## Features
- 2-player local Tic Tac Toe (click to place)
- Custom themed board + pieces (images)
- Winner detection (rows, columns, diagonals)
- Draw detection
- Displays winner/draw message, pauses, and resets
- Shows the current player on screen
